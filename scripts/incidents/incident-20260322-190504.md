## Recommendation (repeated from prior incidents)

The watchdog MUST add an early eth0 carrier check to suppress alerts when the physical link is down:
```bash
if [[ $(cat /sys/class/net/eth0/carrier 2>/dev/null) != "1" ]]; then
    echo "eth0 has no carrier — PLC physically unreachable, skipping alert"
    exit 0
fi
```
This single check would have prevented all three PLC-related watchdog triggers today.
