## Recurring recommendation
The watchdog needs improvements to reduce false alarms:
1. Check for actively-written .prog files, not just completed .capture files
2. Add a grace period after viam-server restarts (suppress alerts for 60s)
3. Filter startup-transient errors from error rate calculations
4. Add hysteresis: require 2 consecutive failed checks before alerting
